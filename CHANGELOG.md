# Changelog

All notable changes to this project will be documented here.

## 0.1.1 - 2026-04-09

- Upgraded Next.js from `14.2.5` to `14.2.35`
- Added a first GitHub release flow with changelog-backed release notes
- Improved public repo health with issue templates, PR template, and code of conduct
- Fixed CI clean-install reliability by removing the unused `ollama-ai-provider` dependency

## 0.1.0 - 2026-04-09

- Initial public release of `standard-ui`
- Added provider-agnostic chat UI with OpenAI-compatible, Anthropic, Ollama, and custom gateway support
- Added public OSS docs, CI, security policy, and contributor guidance

# Changelog

All notable changes to this project will be documented here.

## 0.1.1 - 2026-04-09

- Upgraded Next.js from `14.2.5` to `14.2.35`
- Added release notes and changelog-based version tracking
- Added issue templates, a pull request template, and a code of conduct
- Removed the unused `ollama-ai-provider` dependency to fix `npm ci`

## 0.1.0 - 2026-04-09

- Initial public release of `standard-ui`
- Added provider-agnostic chat UI with OpenAI-compatible, Anthropic, Ollama, and custom gateway support
- Added CI, security policy, and contributor documentation
